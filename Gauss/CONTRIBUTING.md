# Contributing to Gaussian Batch Calculator

Thank you for your interest in contributing to the Gaussian Batch Calculator project! We appreciate contributions of all kinds - from bug reports to feature enhancements.

## How to Contribute

### Reporting Bugs
- Check existing issues to avoid duplicates
- Describe the issue clearly with steps to reproduce
- Include your environment details (OS, Python version, Gaussian version)
- Provide sample input files if possible (but remove any sensitive information)

### Suggesting Features
- Open an issue to discuss your idea before implementing
- Explain the use case and benefits of your proposed feature
- Consider the impact on existing functionality

### Submitting Changes
1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Add tests if applicable
4. Update documentation as needed
5. Submit a pull request with a clear description of your changes

## Code Style
- Follow PEP 8 guidelines for Python code
- Write clear, descriptive comments
- Maintain consistency with existing code style
- Include docstrings for functions and classes

## Questions?
Feel free to open an issue to ask questions about the project or contribution process.

Thank you for helping improve this tool!
